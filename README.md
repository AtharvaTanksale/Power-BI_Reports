# Power BI Portfolio | Financial & Data Analysis

## Overview
Welcome to my Power BI portfolio repository. This space showcases my ability to extract, clean, and model complex datasets to build interactive, professional-grade dashboards. My primary focus is on developing visual analytics that drive strategic decision-making in finance, investment banking, and macroeconomic forecasting.

Each folder in this repository contains a standalone project, complete with the raw data, the `.pbix` file, and a dedicated README detailing the methodologies used.

## Core Competencies & Skills
* **Data Transformation:** Power Query, data cleansing, handling missing values, and type conversion.
* **Data Modeling:** Building relational data models, star schemas, and managing cardinalities.
* **Calculations (DAX):** Writing custom Data Analysis Expressions for rolling averages, year-over-year growth, and aggregate financial metrics.
* **Advanced Visualizations:** Designing executive-level dashboards with custom themes, drill-throughs, dynamic filtering, and tooltip customization.
* **Ecosystem Integration:** Utilizing Python for initial data scraping/modeling before importing clean data into Power BI.

## Featured Projects

### 1. Tech Giant M&A Activity & Deal Volume Dashboard
* **Domain:** Investment Banking / Corporate Strategy
* **Impact:** Visualizes 30 years of acquisition data across top tech firms to identify market consolidation trends and geographic investment footprints.
* **Key Features:** Cross-filtering market share donuts, geographic map plotting, and dynamic average deal value KPIs.


### 2. Macroeconomic Impact Tracker 
* **Domain:** Macroeconomics / Market Research
* **Impact:** Correlates major macroeconomic indicators (inflation rates, federal interest rates) with historical stock market index performance to identify systemic market vulnerabilities.
* **Key Features:** Dual-axis trending, complex DAX time-intelligence functions, and parameter-driven "what-if" scenarios. 


### 3. Predictive Stock Price Visualizer *(Planned)*
* **Domain:** Quantitative Finance
* **Impact:** Translates Python-based machine learning predictions into digestible, executive-facing visual reports comparing historical actuals against forecasted trends.
* **Key Features:** Python visual integration within Power BI, dynamic forecasting bounds, and variance analysis.


### 4.   Player Development Engine Project
* **Doamain:** Sports Analytics / Performance Engineering
* **Impact:** Bridges the gap between raw box scores and coaching insights by filtering out "game-to-game noise" to identify a player's true developmental trajectory and scaling potential.
* **Key Features:**

**Automated Python ETL Pipeline:** Extracts live, game-by-game roster box scores via the nba_api to feed dynamic dashboards.
  
**Advanced DAX Time-Intelligence:** Utilizes 10-game rolling averages to smooth statistical noise and reveal true developmental trajectories.

**Predictive & Correlative Modeling:** Features interactive "What-If" rotational projections and Usage vs. True Shooting (TS%) efficiency analysis.
  

## How to Navigate This Repository
To interact with these reports locally:
1. Clone this repository to your local machine.
2. Ensure you have **Microsoft Power BI Desktop** installed (available for free on Windows).
3. Open any `.pbix` file to interact with the fully cross-filtered dashboards and inspect the underlying DAX measures and Power Query steps.
